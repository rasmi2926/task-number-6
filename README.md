# K-Nearest Neighbors (KNN) Classification

## Objective
Implement the K-Nearest Neighbors (KNN) algorithm for a classification problem using the Iris dataset.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Dataset
Iris Dataset

## Steps Performed
- Loaded the dataset
- Checked missing values
- Removed unnecessary columns
- Split the dataset into training and testing sets
- Normalized features using StandardScaler
- Trained a KNN classifier
- Evaluated the model using accuracy, confusion matrix, and classification report
- Tested different K values
- Visualized Accuracy vs K
- Visualized the decision boundary

## Results
- Achieved high classification accuracy
- Compared model performance for different K values
- Generated confusion matrix and classification report

## Conclusion
The KNN classifier performed well on the Iris dataset. Feature normalization improved distance calculations, and selecting an appropriate K value helped achieve better accuracy.
